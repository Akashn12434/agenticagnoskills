# PowerPoint Slide Design Guide

## Standard Presentation Layout

### Slide 1 — Title

Contains:

- Presentation title
- Subtitle
- Author or organization
- Date (optional)

### Content Slides

Each slide should contain:

- One clear title
- Supporting bullet points
- Visual hierarchy
- Consistent spacing

### Final Slide

Typically Contains:

- Thank You
- Questions
- Contact information (optional)

---
## Content Density

Recommended:

- 3–6 bullet points per slide
- 5–12 words per bullet when possible
- No large paragraphs

Avoid:

- Dense text blocks
- Excessive animations
- Too many colors
- Tiny fonts

---

## Font Guidelines

### Allowed System Fonts
- Primary: Arial, Calibri, Helvetica
- Secondary: Verdana, Tahoma, Times New Roman

### Typography Scale Constraints

| Layout Element | Font Size Target | Font Weight |
|:---|:---|:---|
| Main Presentation Title | 36–44 pt | Bold |
| Section Header / Slide Title | 28–36 pt | Bold |
| Body Content / Bullet Lines | 18–24 pt | Normal |
| Caption / Footer / Source Notes | 10–14 pt | Light / Regular |

## Layout Rules and Recommendations

### Title + Bullets Layout

Use this template configuration for running standard informational text sequences:
- Structural Explanations
- Key Term Definitions
- Bulleted Summaries

### Two-Column Grid Layout

Use this template side-by-side framework configuration for:
- Direct Comparisons
- Advantages vs. Disadvantages (Pros/Cons)
- Case Studies (Before vs. After states)

### Three-Column Grid Layout

Use this template matrix structure for:
- Distinct Categories
- Process Frameworks
- Complex Feature Item Matrices

### Design Guardrails

- **No Chart Stacking**: Never vertically stack charts below text lines within a single slide envelope due to boundary overflow.
- **Maintain Visual Balance**: Avoid overcrowding any slide with excessive content.
- **Consistent Alignment**: Use consistent margins and object alignment throughout the deck.
- **Image Treatment**: Full-bleed images with high-contrast text overlays should be used for title or impact cards.
- **Readability First**: Ensure all text remains legible when projected or viewed on a laptop screen.
